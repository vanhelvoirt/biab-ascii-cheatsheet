# ASCII Cheatsheet (Printable, A4 Landscape)

This repository contains a fully self‑contained HTML document designed for printing multi‑page ASCII reference sheets.  
It is optimized for A4 landscape, monospace formatting, and manual page breaks, making it ideal for BIAB workflows, Reaper shortcuts, studio notes, or any structured ASCII layout.

## Features

- 4‑page layout ready for ASCII content  
- A4 landscape print mode enforced via CSS  
- Monospace font for perfect ASCII alignment  
- Manual page breaks using <div class="pagebreak"></div>  
- Print button included (hidden during printing)  
- Metadata: author, version, last updated, keywords  
- Single‑file HTML — no external CSS required  

## How to Use

1. Open `cheatsheet.html` in any modern browser (Chrome, Edge, Firefox).
2. Locate the four `<pre class="page">` sections.
3. Paste your ASCII content into each section.
4. Click **Print this document** at the bottom.
5. The browser will automatically switch to A4 landscape mode.

## Printing Notes

- Print button is hidden during printing.
- Font size is set to 12pt for optimal readability.
- Page breaks are controlled manually using `<div class="pagebreak"></div>`.
- The file must be opened in a real browser window, not an editor preview.

## Customization

You can modify:
- Number of pages  
- Font size  
- Margins  
- Metadata (author, version, keywords)  
- Headers or footers per page
